Informações sobre o Comandos Básicos


Para os comandos básicos, vamos tratar apenas de comandos que são utilizados no dia-a-dia.

Comando Descrição Imagem de exemplo
CD Ele permite que você navegue entre diretórios e acesse diferentes localizações no sistema. Comando Cd
LS Listar arquivos ou pastas em que o usuário esteja localizado Comando Ls
Exit Saida do Terminal do Linux
PWD Mostra o caminho que se encontra para chegar a posição atual Comando Pwd
Clear Limpa o terminal deixando na primeira linha Comando Clear
MAN Lista arquivos ou pastas em que o usuário esteja localizado Comando Man
